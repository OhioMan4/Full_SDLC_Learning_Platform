# API

Core endpoints:

- `GET /health`
- `GET /courses`
- `GET /courses/{course_id}`
- `GET /courses/{course_id}/lessons`
- `POST /courses`
- `POST /courses/{course_id}/lessons`
- `POST /enrollments`
- `GET /enrollments`
- `GET /enrollments/{enrollment_id}`
- `GET /students/{student_id}/enrollments`
- `POST /enrollments/{enrollment_id}/cancel`
- `POST /progress/lessons/complete`
- `GET /students/{student_id}/courses/{course_id}/progress`
- `GET /students/{student_id}/progress`
- `GET /students/{student_id}/certificates`
- `GET /certificates/{certificate_id}`
- `GET /students/{student_id}/notifications`
- `GET /audit-events`
